User flagged that answers repeating set phrases ("The payoff is the triangulation", "Methodological hygiene to keep clean") and leading paragraphs with abstract label-intros makes them harder to read.

**Why:** dilutes signal; the user reads dense technical proposals and wants the substance, not framing scaffolding.

**How to apply:** state the point directly instead of announcing it with a labeled lead-in. Don't reuse the same framing phrase across a reply. Prefer fewer words. Cut recap sentences that restate what was just said.

User flagged again (2026-06-10): an entire stock paragraph ("the confluence trap" — a tradeoff/continuum framing) recurred across answers; reads like a bug and creates confusion in an otherwise solid reply. Don't reach for a canned conceptual block — write only to the actual question asked. See also [[feedback_no_mean_reversion_framing]].

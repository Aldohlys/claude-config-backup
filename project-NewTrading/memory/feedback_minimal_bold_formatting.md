User finds heavy **bold** tiring and useless: "Putting in bold expressions does not help me to get the meat of the trade." Structure should come from sections / sub-sections, not sprinkled emphasis.

**Rule for bold:**
- KEEP bold only when it is the **leading label of a list item or paragraph that then gets detailed** — e.g. "**FOMO entries** — GLD, SLV…" in a What-fails list, step labels ("**Work the mid.** Place the limit…"), `Source:`-style intro labels, and table first-column row labels.
- REMOVE bold used for inline emphasis inside a sentence — e.g. do NOT bold "specific type" in "The BOT screener captures a specific type of breakout", or "different timeframes", "one bet not ten", etc.
- Fully-bold standalone sentences (emphasis, not a label) → unbold.

**Why:** durable writing-style preference. **How to apply:** write docs/reports this way from the start; default to plain prose + clear section headings. Applied 2026-06-15 to `bot_strategy_trading_plan.md` + `bot_short_strategy_trading_plan.md` (stripped ~95 inline-bold spans, kept labels).

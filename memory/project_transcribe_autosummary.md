**Transcribe pipeline** lives in `NewTrading/` (a working dir, separate git repo):
- Launcher: `scripts/launcher.py` (Tkinter) auto-discovers `*.bat`; "run transcribe" = `scripts/run_transcribe.bat` → `Transcripts/Run-Transcribe.ps1`.
- Pipeline: yt-dlp download → `Transcripts/transcribe.py` (Whisper large-v3) → `wherestrade_<YYYYMMDD>_transcript.txt`.

**2026-06-02:** added an auto-summary step (Run-Transcribe.ps1 Step 5). The summary was previously a MANUAL Claude upload; now the pipeline runs `claude -p "/transcribe <transcript>" --permission-mode acceptEdits` (cwd=Transcripts) → writes `wherestrade_<date>_summary.md`. The summary is LLM-generated by the project `/transcribe` command (P5/P6 in `NewTrading/.claude/commands/transcribe.md`), not a deterministic script. `-SkipSummary` switch + non-fatal guards. See [[reference_claude_headless_slash_command]].
